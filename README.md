## Getting Started

To get started, simply clone the repository and run `npm install`

```
# Clone the repo
https://github.com/nolawi/meteor

# Move into the new directory
cd meteor/

# Install npm packages
npm install

# Start up browse to http://localhost:8080/
npm start
```
